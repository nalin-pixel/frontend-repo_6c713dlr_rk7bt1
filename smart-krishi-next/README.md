# Smart Krishi (Next.js + Tailwind + Vercel)

Production-ready template with:
- Pages: Home, Features, How It Works, Demo, Pricing, Testimonials, Contact
- Bilingual support (EN/HI)
- Camera/Gallery upload (demo)
- Serverless APIs: disease detection, weather, mandi, fertilizer, demo
- Tailwind configured out-of-the-box

## Getting Started

1. npm install
2. npm run dev
3. Open http://localhost:3000

## Environment

- OPENWEATHER_API_KEY (optional for live weather)

## Deploy

- vercel login
- vercel
- vercel --prod
